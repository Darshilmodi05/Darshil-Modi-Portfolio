## 🚀 Quick Start

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional for development)

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Darshilmodi05/Darshil-Modi-Portfolio.git
   cd Darshil-Modi-Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   - Navigate to `http://localhost:3000`
   - Or open `index.html` directly in your browser

## 🌐 Deployment

### Automatic Deployment
The portfolio is automatically deployed to Vercel when pushing to the main branch.

### Manual Deployment
1. Push changes to GitHub
2. Vercel will automatically detect and deploy
3. Access your portfolio at the provided Vercel URL

### Custom Domain
To use a custom domain:
1. Purchase a domain from your preferred registrar
2. Add domain in Vercel dashboard
3. Update DNS records as instructed by Vercel
