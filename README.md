## 📋 Project Overview

Welcome to the source code repository for my professional portfolio website. This application serves as a comprehensive digital resume, highlighting my experience, technical capabilities, and software engineering projects. It is architected to deliver a high-performance, accessible, and visually engaging user experience.

🌐 **Live Application:** [darshilmodi.vercel.app](https://darshilmodi.vercel.app)

## 🛠️ Technical Architecture

This portfolio is built with a focus on modern web standards, lightweight performance, and robust maintainability.

### Core Stack
- **Structure:** Semantic HTML5 ensuring high accessibility standards and SEO optimization.
- **Design System:** Custom CSS3 implementing a mobile-first responsive design, dark/light theme toggling, and hardware-accelerated animations.
- **Logic & Interactivity:** Modular ES6+ JavaScript handling DOM manipulation, form validation, and complex canvas-based particle rendering.
- **Infrastructure:** Configured for seamless continuous integration and deployment (CI/CD) via **Vercel**.

## ⚙️ Local Development Environment

To review the source code or run the application locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Darshilmodi05/Darshil-Modi-Portfolio.git
   cd Darshil-Modi-Portfolio
   ```

2. **Install local dependencies:**
   ```bash
   npm install
   ```

3. **Initialize the local server:**
   ```bash
   npm run dev
   ```

4. **View application:**
   Access `http://localhost:3000` via any modern web browser.
