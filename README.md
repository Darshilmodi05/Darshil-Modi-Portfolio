## 📋 Project Overview

Welcome to the source code repository for my professional portfolio website. This application serves as a comprehensive digital resume, highlighting my experience, technical capabilities, and software engineering projects. It is architected to deliver a high-performance, accessible, and visually engaging user experience.

🌐 **Live Application:** [darshil-modi.vercel.app](https://darshil-modi.vercel.app)

## 🛠️ Technical Architecture

### Core Stack
- **Structure:** Semantic HTML5 ensuring high accessibility standards and SEO optimization.
- **Design System:** Custom CSS3 implementing a mobile-first responsive design, dark/light theme toggling, and hardware-accelerated animations.
- **Logic & Interactivity:** Modular ES6+ JavaScript handling DOM manipulation, form validation, and canvas-based particle rendering.
- **Infrastructure:** Configured for seamless CI/CD via **Vercel**.

## 🚀 Key Engineering Projects

- **Medora Health Platform** — Full-stack healthcare ecosystem with Next.js, FastAPI & Neon DB. Features role-based access and AI-assisted diagnostics.
- **Skin AI Scanner** — ML pipeline using Python & TensorFlow to classify dermatological lesions with high accuracy.
- **Zaika Palace** — Enterprise-grade restaurant management system on the MERN stack.
- **AI Chat Assistant** — Conversational interface integrating OpenAI LLMs with a React frontend and Python backend.

## ⚙️ Local Development

```bash
# 1. Clone the repository
git clone https://github.com/Darshilmodi05/Darshil-Modi-Portfolio.git
cd Darshil-Modi-Portfolio

# 2. Install dependencies
npm install

# 3. Start local server
npm run dev
```

Access the application at [http://localhost:3000](http://localhost:3000).
